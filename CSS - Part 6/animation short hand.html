Animation Shorthand

Use one line: animation: name duration timing delay count direction;

Definition + Use Case

  • Definition: animation shorthand combines many animation properties in one line.
  • Order (common): animation: name duration timing-function delay iteration-count direction;
  • Example meaning: jump 1.2s ease-in-out 0s infinite alternate = run jump for 1.2s, smooth speed, no delay, repeat forever, go forward/backward.
  • Use case: Write cleaner CSS and avoid repeating separate animation properties.
  • Tip: Duration and name are most important; other values can be added as needed.
