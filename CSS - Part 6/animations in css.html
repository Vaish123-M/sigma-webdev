Animations in CSS

Animation needs @keyframes + animation properties.

Property Definitions + Use Case

  • animation-name: Selects which @keyframes to run. Use case: Reuse different motion styles like slide, fade, bounce.
  • animation-duration: Time for one full cycle. Use case: Keep micro-animations short (0.2s–0.6s), loaders longer.
  • animation-timing-function: Controls speed pattern (linear, ease, etc.). Use case: Natural UI motion with ease-in-out.
  • animation-delay: Wait time before start. Use case: Stagger multiple cards/buttons for better visual sequence.
  • animation-iteration-count: Number of repeats (1, 3, infinite). Use case: Infinite for loaders, single run for intro effects.
  • animation-direction: Playback direction (normal, reverse, alternate). Use case: alternate for smooth back-and-forth motion.
  • Equivalent shorthand: Instead of writing all separate properties, you can write one line like animation: widthAnimate 3s ease-in 0s infinite normal; (same effect if values match).
